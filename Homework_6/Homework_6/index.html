border

solid - суцільна лінія

dotted - пунктирна лінія

dashed - штрихована лінія

double - подвійна лінія

transparent - прозорий колір

комбіновані лінії

outline - обводка навколо елемента

outline-offset - відстань від об'єкта до обводки

border-radius - радіус заокруглення кутів блоку

border-radius: 10px;

border-radius: 20px 10px 5px 0px

circle

border-radius: 50%;

box-shadow

box-shadow: 5px 2px 10px 10px rgba;

inset - тінь в середені об'єкту

opacity

opacity: 0 - повна прозорість, з об'єктами що не бачимо, взаємодіяти можемо, займає місце на сторінці

opacity: 0;

opacity: 0.5;

opacity: 1;

visibility

visibility: hidden - об'єкт ховається, з ним не можна взаємодіяти, займає місце на сторінці

visibility: hidden;

visibility: visible; за замовченням

display: none

display: none - об'єкт ховається, з ним не можна взаємодіяти, він не займає місце на сторінці

background-color

background-color: #c1c5c0;

background-image

background-gradient

background-repeat

background-repeat: no-repeat; - Не повторяти

background-repeat: repeat-x; - Повтор по горизонталі

background-repeat: repeat-y; - Повтор по вертикалі

background-repeat: repeat; - Повтор по горизонталі та вертикалі

background-position

background-position: відсотки, px, слова; - Визначає положення фону

background-attachment

background-attachment: fixed; - Фон фіксується відносно в'юпорта

background-size

background-size: contain; - Масштабує зображення зі збереженням пропорцій

background-size: cover; - Масштабує зображення до заповнення блоку

background-clip

background-clip: padding-box; - Фон обрізається по кордону padding

background-clip: text; - Фон обрізається по тексту
