width

width - чітко задає ширину блочного (блочно-рядкового) об'єкту.

max-width - Встановлює максимальну ширину блочних (блочно-рядкових) тегів

min-width - Встановлює мінімальну ширину блочних (блочно-рядкових) тегів

height

height - чітко задає висоту блочного (блочно-рядкового) об'єкту.

max-height - Встановлює максимальну висоту блочних (блочно-рядкових) тегів

min-height - Встановлює мінімальну висоту блочних (блочно-рядкових) тегів

overflow

overflow: visible - Відображається усесь контент елемента, навіть за межами встановлених розмірів.

overflow: hidden - Відображається тільки контент всередині елемента.
